# Support

AIppliance-Magic-Stick is an open-source template. Support is provided on a
best-effort basis by maintainers and contributors.

## Where To Ask

- Use GitHub issues for reproducible bugs, documentation problems, and feature
  requests that are safe to discuss publicly.
- Use GitHub discussions if they are enabled for general questions.
- For suspected vulnerabilities, leaked credentials, or deployment-specific
  details, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

## Public Safety

Do not include real domains, kubeconfigs, tokens, private repository paths,
customer names, screenshots with secrets, or decoded Kubernetes Secret values in
public support requests.

Use placeholders such as `example.com`, `example.local`, `CHANGEME`, and
redacted log excerpts.

## Response Expectations

The project does not currently offer commercial support or guaranteed response
times from this repository. Maintainers may prioritize reports that include:

- the exact commit or release tag
- rendered Kustomize output or failing command output
- Kubernetes resource status with sensitive values removed
- clear reproduction steps
