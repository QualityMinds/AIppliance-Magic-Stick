# Third-Party Notices

This repository does not vendor third-party source code, container images, or
Helm charts. It references upstream artifacts that are pulled at deployment time.
Review upstream licenses, image terms, and trademarks before publishing a release
or using the template in a commercial deployment.

License information below is a release-review aid, not legal advice. Because most
artifacts are pulled from upstream registries at deployment time, each release
should verify the current upstream license and image terms for the pinned tag,
digest, or chart version.

## Release Review

Before a public release tag, confirm that referenced third-party artifacts are
acceptable for the intended use, that mutable tags have been avoided where
practical, and that brand names are used only to identify integrations.

Release reviewers should record:

- upstream license or terms for each pinned artifact
- whether an artifact is referenced, bundled, mirrored, or redistributed
- chart and image version changes since the previous release
- any trademark-sensitive wording in docs, screenshots, or landing pages
