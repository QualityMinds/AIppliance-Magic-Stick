A self-hosted AI appliance for local models, agents, apps, and Kubernetes-native automation. Bootstrap a machine, let Flux converge the platform, then manage modules, model activations, app instances, domains, and ingress links from one dashboard.

Build a small AI platform without turning it into a platform team.

MagicStick packages the boring but critical pieces of a local AI appliance: host bootstrap, K3s, Flux, GPU platform bits, model serving, app operators, ingress naming, and a dashboard that writes runtime CRs instead of hand-editing cluster state.

Dashboard-managed runtime

Enable modules, add local or external models, create app instances, inspect progress, and open generated ingress URLs.

Catalog-driven modules

The module catalog controls display names, dependencies, activation mode, parameters, and instance availability.

Public-safe base repo

Keep this repository reusable and generic while private deployments provide real domains, tokens, and overlays.

Local model workflows

Deploy KubeAI-backed models, surface catalog status, and estimate VRAM from HuggingFace metadata before creating activations.

Public and mDNS ingress

Publish apps through predictable host schemes like instance.type.domain and local `.local` names.

GitOps as the baseline

Flux owns the platform graph, while runtime resources let users work through Kubernetes APIs and the dashboard.

Start from the public template, then layer private deployment values.

The public repo stays safe to share. Real domains, credentials, and site-specific overrides belong in runtime settings or a private GitOps repository that includes this template.

  1. Clone the template Use the public repository as the reusable base for appliance bootstrap and cluster rendering.
  2. Render the Flux graph Validate the public base or the single-node profile before applying it to a cluster.
  3. Converge the appliance Flux applies modules, the MagicStick Operator reconciles CRs, and the dashboard exposes runtime actions. 
git clone https://github.com/QualityMinds/AIppliance-Magic-Stick
cd AIppliance-Magic-Stick

kubectl kustomize magic-cluster/flux/entrypoints/base
kubectl kustomize magic-cluster/flux/entrypoints/single-node
kubectl kustomize magic-cluster/apps/dashboard
