# Code of Conduct

This project should be a practical, respectful place to collaborate.

## Expected Behavior

- Be kind, direct, and constructive.
- Assume good intent while still naming risks clearly.
- Keep technical criticism focused on code, documentation, behavior, and safety.
- Respect boundaries around private deployment data and security-sensitive details.

## Unacceptable Behavior

- Harassment, threats, intimidation, or discriminatory language.
- Publishing private information, credentials, or deployment details.
- Repeatedly derailing technical discussions after maintainers have redirected them.
- Using project spaces for spam or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or remove comments and may limit participation when
behavior makes collaboration unsafe or unproductive. Security issues and private
deployment details should be moved to private channels immediately.
