# Roadmap

This roadmap captures likely public project directions. It is not a commitment
to deliver specific features on a fixed date.

## Near Term

- Keep dashboard, module, instance, and model documentation aligned with runtime
  CR behavior.
- Improve release hygiene around third-party notices and pinned artifacts.
- Expand validation examples for public read-only installs.

## Later

- Replace implementation notes with generated or versioned API references where
  possible.
- Clarify compatibility expectations for downstream external deployment overlays.
- Consider a typed controller implementation once the public CRD contracts have
  stabilized.

## Out Of Scope

- Committing real deployment values, credentials, private repo paths, or customer
  configuration to this public repository.
- Providing guaranteed production support from the public issue tracker.
